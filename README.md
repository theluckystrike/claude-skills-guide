# Claude Skills Guide

700+ free tutorials for Claude Code skills -- comparisons, troubleshooting, integrations, and automation guides.

**Browse:** [theluckystrike.github.io/claude-skills-guide](https://theluckystrike.github.io/claude-skills-guide/)

### What's inside

- **Getting Started** -- Installation, first skills, SKILL.md format
- **Comparisons** -- Claude Code vs Cursor, Copilot, Windsurf, and more
- **Integrations** -- Docker, Git, CI/CD, databases, 50+ tools
- **Troubleshooting** -- Common errors, debugging, performance
- **Advanced** -- Multi-agent workflows, custom tooling, automation

### Built with

Jekyll on GitHub Pages. Articles are standalone Markdown files in `articles/`.
